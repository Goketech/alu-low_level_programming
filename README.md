# ALU Low-Level Programming

This repository contains introductory low-level programming projects focused on C, compilation stages, and basic Linux tooling.

## Project Structure

- ` hello_world/`: A collection of beginner tasks that introduce the C compilation pipeline and fundamental C output programs.

## Main Goals

- Understand what happens when C code is preprocessed, compiled, assembled, and linked.
- Practice writing simple C programs using different output functions.
- Learn to inspect binary and architecture information.
- Work with executable scripts and source files in a Unix-like environment.

## Requirements

- Ubuntu/Linux environment (or compatible shell)
- `gcc`
- `bash`/`sh`

## Notes

- Files inside the ` hello_world/` directory are set with user executable permission (`u+x`) as requested.
